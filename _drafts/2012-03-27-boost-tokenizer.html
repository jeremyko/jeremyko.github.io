boost::tokenizer

#include <boost/tokenizer.hpp>
#include <string>
#include <iostream>

int main()
{
   typedef boost::tokenizer<boost::char_separator<char> > tokenizer;
   std::string s = "Boost C++ libraries";
   tokenizer tok(s); // std::string으로 초기화 되어야함.
   for (tokenizer::iterator it = tok.begin(); it != tok.end(); ++it)
     std::cout << *it << std::endl;
}

boost::char_separator는 기본적으로 공백과 구두점을 구분자로 해석한다
(이러한 문자를 구분하기 위해 std::isspace() 과 std::ispunct()가 사용된다).
boost::tokenizer는 표시되어야할 구분자와 표시되지 말아야할 구분자를 구별하는데,
기본값은 공백은 표시안됨이고 구두점은 표시하는 것이다.
그래서 위의 예제를 실행하면 출력은 다음과 같다.
Boost
C
+
+
libraries


만약 구두점을 구분자로 사용하지 않으려면
boost::char_separator 을 다른방법으로 초기화 해야한다.

#include <boost/tokenizer.hpp>
#include <string>
#include <iostream>

int main()
{
   typedef boost::tokenizer<boost::char_separator<char> > tokenizer;
   std::string s = "Boost C++ libraries";
   boost::char_separator<char> sep(" ");
   tokenizer tok(s, sep);
   for (tokenizer::iterator it = tok.begin(); it != tok.end(); ++it)
     std::cout << *it << std::endl;
}

boost::char_separator 생성자는 3개의 인자가 필요한데,
첫번째 인자는 필수이고, 표시되지 않는 구분자를 나타내며, 나머지는 선택사항이다.
위 예제에서 공백이 구분자로 사용되고, 화면에는 출력되지 않을것이다.
두번째 인자는 역시 구분자인데, 이것은 화면에 표시되는 구분자를 지정한다.
생략된다면 어떠한 구분자도 화면에 출력되지 않는다.
따라서 위 예제를 실행하면 출력은 다음과 같다.
Boost
C++
libraries

만약 첫번째 예처럼 +기호가 출력이 되게 하려면, 두번째 인자로 지정하면 된다.즉,
int main()
{
   ...
   boost::char_separator<char> sep(" ", "+");
   tokenizer tok(s, sep);
   ...
}

세번째 인자는 구분자가 연달아 있는경우, 빈문자열을 출력할지 여부를 결정한다
(default 동작은 출력안됨이다).

int main()
{
   typedef boost::tokenizer<boost::char_separator<char> > tokenizer;
   std::string s = "Boost C++ libraries";
   boost::char_separator<char> sep(" ", "+", boost::keep_empty_tokens);
   tokenizer tok(s, sep);
   for (tokenizer::iterator it = tok.begin(); it != tok.end(); ++it)
     std::cout << *it << std::endl;
}

위 예제를 실행하면 결과는 다음과 같다.
Boost
C
+

+

libraries

++ 사이의 빈 문자열, 그리고 +와 그다음 공백으로 인한 빈 문자열이 출력된다.

또한 tokenizer는 상이한 string타입에 대해서 사용가능하다.
#include <boost/tokenizer.hpp>
#include <string>
#include <iostream>

int main()
{
   typedef boost::tokenizer< boost::char_separator<wchar_t>, 
   std::wstring::const_iterator, std::wstring
   > tokenizer; //추가 인자
   std::wstring s = L"Boost C++ libraries";
   boost::char_separator<wchar_t> sep(L" ");
   tokenizer tok(s, sep);
   for (tokenizer::iterator it = tok.begin(); it != tok.end(); ++it)
     std::wcout << *it << std::endl;
}

boost::char_separator 외, 2가지 추가적인 클래스가 제공됨.

boost::escaped_list_separator는 콤마로 구분된 구분자를 지정할수 있다.
또한 "과 \도 구분자로 인식한다.

#include <boost/tokenizer.hpp>
#include <string>
#include <iostream>

int main()
{
   typedef boost::tokenizer<boost::escaped_list_separator<char> > tokenizer;
   std::string s = "Boost,\"C++ libraries\"";
   tokenizer tok(s);
   for (tokenizer::iterator it = tok.begin(); it != tok.end(); ++it)
     std::cout <<"["<< *it << "]"<<std::endl;
}

출력:
[Boost]
[C++ libraries]

boost::offset_separator 는 반드시 초기화가 필요하며, 문자열에서 각 부분문자가
끝나는 위치를 지정한다.

#include <boost/tokenizer.hpp>
#include <string>
#include <iostream>

int main()
{
   typedef boost::tokenizer<boost::offset_separator> tokenizer;
   std::string s = "Boost C++ libraries";
   int offsets[] = { 5, 5, 9 };
   boost::offset_separator sep(offsets, offsets + 3);
   tokenizer tok(s, sep);
   for (tokenizer::iterator it = tok.begin(); it != tok.end(); ++it)
     std::cout <<"["<< *it << "]"<<std::endl;
}

위 예제에서, 5번째, 그리고 그이후 다시 5번째, 이후 9번째를 구분자로 지정했고
출력은 다음과 같다.
[Boost]
[ C++ ]
[libraries]
