--- layout: post title: c++ trivial 기본 생성자(작성자료) date: '2017-02-13T23:49:00.000+09:00' author: jeremyko tags: - c++ - trivial modified_time: '2017-02-13T23:49:14.307+09:00' blogger_id: tag:blogger.com,1999:blog-7360229670252766698.post-7878223671121460635 blogger_orig_url: https://jeremyko.blogspot.com/2017/02/c-trivial.html --- trivial : 사전적인 의미로 보면 중요하지않은, 사소한... 이런 의미가 있는데 c++ 에서 의미는 신경 쓸 필요없다.. 중요하지 않다 .. 이런 의미다.


예를 들면 어떤 객체가 있는데, 생성자, 소멸자를 사용자가 정의하지않아서 컴파일러가 암묵적으로 기본 생성자를 생성한 경우, 다음 조건을 만족하면 trivial 생성자라고 한다
  • 가상함수나 베이스 가상함수가 없음
  • 비 정적 데이터 멤버가 brace-or-equal-initializer 가지면 안됨
  • 부모 클래스의 기본 생성자도 trivial 해야 함.
  • non static 멤버 중에 class 타잎인 것의 모든 기본 생성자도 trivial 해야함.

반대로 말하면 자신이 명시적으로 생성자를 정의한다면(비록 아무일도 안하는 기본 생성자 일지라도) 그 생성자는 non-trivial 하다.
