boost::any
모든 종류의 정보를 저장할수 있다 (JavaScript 에서의 Variables) .

#include <boost/any.hpp>

int main()
{
   boost::any a = 1;
   a = 3.14;
   a = true;
}

여기서 주의할점은, boost::any는 한가지 전제조건을 요구하는데, 즉 복사 생성자를 지원해야 한다는 것이다. 
그런 이유로 문자 스트링을 저장하기 위해서는 그냥 대입은 안되고, 다음에서처럼 std::string를 명시적으로 사용할 필요가 있다.



#include <boost/any.hpp>
#include <string>

int main()
{
   boost::any a = 1;
   a = 3.14;
   a = true;
   a = std::string("Hello, world!");

   //a = "Hello, world!";  => 에러.. not copy constructible!!
}

저장된 데이터에 접근 하기 위해서는 boost::any_cast 를 사용한다.

#include <boost/any.hpp>
#include <iostream>

int main()
{
   boost::any a = 1;
   std::cout << boost::any_cast<int>(a) << std::endl;
   a = 3.14;
   std::cout << boost::any_cast<double>(a) << std::endl;
   a = true;
   std::cout << boost::any_cast<bool>(a) << std::endl;
}

boost::any_cast에 템플릿 인자로 적절한 데이터형을 전달하면, 그 타입으로 변환된 결과를 얻을수 있다. 
만약 잘못된 데이터 타입을 지정한다면, boost::bad_any_cast 예외가 발생할 것이다.

#include <boost/any.hpp>
#include <iostream>

int main()
{
   try
   {
         boost::any a = 1;
         std::cout << boost::any_cast<float>(a) << std::endl;
   }
   catch (boost::bad_any_cast &e)
   {
         std::cerr << e.what() << std::endl;
   }
}

int 타입에 대해서 잘못된 타입을 지정하는 경우이다. short 나 long으로 지정해도 마찬가지로 예외가 발생할것이다.
boost::bad_any_cast 는 std::bad_cast 로부터 상속되었다.

boost::any 타입 변수가 정보를 가지고 있는지 여부를 확인하기 위해서 empty() 메서드를 사용한다. 
저장된 데이터의 타입을 알고 싶은 경우에는 type() 를 대신 사용한다.

#include <boost/any.hpp>
#include <typeinfo>
#include <iostream>

int main()
{
   boost::any a = 1;
   if (!a.empty())
   {
     const std::type_info &ti = a.type();
     std::cout << ti.name() << std::endl;
   }
}

다음예는 boost::any에 저장된 값에 대한 포인터를 얻기 위해 boost::any_cast를 사용하고 있다.

#include <boost/any.hpp>
#include <iostream>

int main()
{
   boost::any a = 1;
   int *i = boost::any_cast<int>(&a);
   std::cout << *i << std::endl;
}

boost::any_cast 의 템플릿 인자는 변경이 없고, 단지 boost::any 타입 변수의 포인터를 입력인자로 전달하면 된다.
