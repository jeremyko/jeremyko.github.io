boost::lambda

이름없는 함수를 람다함수라고 부른다.
현 시점에서는 C++표준으로 람다함수가 채택되었지만, boost에 기존부터 존재하던 기능.


#include <iostream>
#include <vector>
#include <algorithm>

void print(int i)
{
std::cout << i << std::endl;
}

int main()
{
   std::vector<int> v;
   v.push_back(1);
   v.push_back(3);
   v.push_back(2);

   std::for_each(v.begin(), v.end(), print);
}

위 예에서 별도로 print함수없이 람다함수를 이용해서 동일하게 처리하고 싶다면

#include <boost/lambda/lambda.hpp>
#include <iostream>
#include <vector>
#include <algorithm>

int main()
{
   std::vector<int> v;
   v.push_back(1);
   v.push_back(3);
   v.push_back(2);

   std::for_each(v.begin(), v.end(), std::cout << boost::lambda::_1 << "\n");
}

boost bind와 마찬가지로 _1 과 같은 placeholder가 존재한다.
하지만 namespace가 클리기때문에 boost::lambda::_1 처럼 사용해야 한다.
예제에서 줄바꿈을 위해서는 std::endl대신 "\n"이 사용되어야 한다.

그밖에도 람다함수내에서 조건식을 위한
boost::lambda::if_then, boost::lambda::if_then_else(),
boost::lambda::if_then_else_return() 등을 제공한다.

#include <boost/lambda/lambda.hpp>
#include <boost/lambda/if.hpp>
#include <iostream>
#include <vector>
#include <algorithm>

int main()
{
   std::vector<int> v;
   v.push_back(1);
   v.push_back(3);
   v.push_back(2);

   std::for_each(v.begin(), v.end(),
  
   //1보다 큰경우만 출력
       boost::lambda::if_then( boost::lambda::_1 > 1,
               std::cout << boost::lambda::_1 << "\n"));
}
