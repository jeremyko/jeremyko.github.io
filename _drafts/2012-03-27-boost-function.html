boost::function

함수포인터. 함수 혹은 함수객체(function object)를 저장할수 있다.
리턴값이  int, 입력인자가 const char* 인 함수포인터는 다음과 같이 선언한다.
boost::function<int (const char*)> f ;


#include <boost/function.hpp>
#include <iostream>
#include <cstdlib>
#include <cstring>

int someFunction1(const char*)
{
return 1;
}

std::size_t someFunction2(const char*)
{
return 2;
}

int main()
{
boost::function<int (const char*)> f = &someFunction1;
//&는 Microsoft Visual C++ version 6사용할때만 꼭 필요하다

std::cout << f("1609") << std::endl;
   f = someFunction2; //리턴타입이 std::size_t 이지만 가능하다.
   std::cout << f("1609") << std::endl;
}

함수 객체인 경우 operator()() 오버로드가 필요하다.
struct someFunctionObj1 {
  int operator()(const char*) const { return 1; };
};

boost::function 타입에 0을 할당하면 기존 설정된 함수는 해제된다. 그리고
이후에 boost::function 을 사용하면 boost::bad_function_call 예외가 발생된다.
현재 설정된 함수 여부를 확인하기 위해서는 empty() 함수나 bool() 연산자를 사용한다.

멤버함수를 설정할수도 있다. 이때는 첫번째 인자로, 클래스형을 포인터형식으로 알려줘야한다.

#include <boost/function.hpp>
#include <iostream>

struct world
{
   void hello(std::ostream &os)
   {
         os << "Hello, world!" << std::endl;
   }
};

int main()
{
   boost::function<void (world*, std::ostream&)> f = &world::hello; //world* !!
   world w;
   f(&w, boost::ref(std::cout));
}
