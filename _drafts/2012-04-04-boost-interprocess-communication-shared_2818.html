Synchronization

공유메모리는 여러 어플리케이션으로부터 접근되므로 동기화수단이 필요하다.
기존에 설명된 Thread를 위한 동기화 방법은 해당 어플리케이션 내에서만 유효하므로
사용할수없다. 2가지의 동기화 객체가 제공된다.
- Anonymous objects : 공유메모리 내에 저장되고 모든 어플리케이션들이 자동적으로 사용가능.
- Named objects : 운영체계에 의해 관리되며 공유메모리내에 존재 안함.
  어플리케이션에서 이름을 가지고 얻어서 사용.

아래 예제에서는 boost::interprocess::named_mutex 의 named mutex object를
생성, 사용하고 있다.

#include <boost/interprocess/managed_shared_memory.hpp>
#include <boost/interprocess/sync/named_mutex.hpp>
#include <iostream>

int main()
{
   boost::interprocess::managed_shared_memory
   managed_shm(boost::interprocess::open_or_create, "shm", 1024);
   int *i = managed_shm.find_or_construct<int>("Integer")();
   boost::interprocess::named_mutex named_mtx(boost::interprocess::open_or_create, "mtx");
   named_mtx.lock();
   ++(*i);
   std::cout << *i << std::endl;
   named_mtx.unlock();
}

mutex object 생성시, 생성/오픈 여부와 함께 mutex를 위한 이름이 필요하다.
그이름을 알고있는 모든 어플리케이션은 동일한 mutex 객체를 얻을수있다.
공유메모리 객체에 대한 접근을 위해서 mutex 객체의 소유권을 얻어야 하는데
이때 lock()을 사용한다. 다른 어플리테이션은 unlock()이 호출될때까지 대기하게 된다.

다음은 anonymous mutex object 를 사용한 예제이다.

#include <boost/interprocess/managed_shared_memory.hpp>
#include <boost/interprocess/sync/interprocess_mutex.hpp>
#include <iostream>

int main()
{
   boost::interprocess::managed_shared_memory
   managed_shm(boost::interprocess::open_or_create, "shm", 1024);
   int *i = managed_shm.find_or_construct<int>("Integer")();

   boost::interprocess::interprocess_mutex *mtx =
   managed_shm.find_or_construct<boost::interprocess::interprocess_mutex>("mtx")();

   mtx->lock();
   ++(*i);
   std::cout << *i << std::endl;
   mtx->unlock();
}

boost::interprocess::interprocess_mutex 객체가 선언되었고 공유메모리내에 저장된다.
앞의 예제와의 차이점은 construct() 나 find_or_construct() 메서드를 이용해서
공유메모리내에 생성된다는 것이다.

lock()메서드 외에 try_lock() 과 timed_lock() 메서드가 제공된다.
(이것들은 Boost.Thread 의 그것과 동일하게 동작한다)
그밖에 recursive 한 mutex 객체가 필요한 경우를 위해 boost::interprocess::named_recursive_mutex,
boost::interprocess::interprocess_recursive_mutex 가 제공된다.

Boost.Thread 와 마찬가지로 conditional variables도 제공된다. 사용법은 거의 동일하다.
( 앞서 설명된 Boost.Thread 동기화 내용은 여기를 클릭)

#include <boost/interprocess/managed_shared_memory.hpp>
#include <boost/interprocess/sync/named_mutex.hpp>
#include <boost/interprocess/sync/named_condition.hpp>
#include <boost/interprocess/sync/scoped_lock.hpp>
#include <iostream>

int main()
{
   boost::interprocess::managed_shared_memory
   managed_shm(boost::interprocess::open_or_create, "shm", 1024);
   int *i = managed_shm.find_or_construct<int>("Integer")(0);
   boost::interprocess::named_mutex named_mtx(boost::interprocess::open_or_create, "mtx");

   boost::interprocess::named_condition
   named_cnd(boost::interprocess::open_or_create, "cnd");

   boost::interprocess::scoped_lock<boost::interprocess::named_mutex> lock(named_mtx);

   while (*i < 10)
   {
     if (*i % 2 == 0)
     {
       ++(*i);

       named_cnd.notify_all(); //조건변수를 통해서 wait가 종료됬음을 알려준다.

       named_cnd.wait(lock);
       //mutex는 자동으로 해제되서,다른 어플이 획득가능하다.
       //그리고 다른 어플이 값을 1증가시켜, 짝수로 만든후,
       //notify_all()을 호출할때까지 대기한다.
     }
     else
     {
       std::cout << *i << std::endl;
       ++(*i);
       named_cnd.notify_all();
       named_cnd.wait(lock);
     }
}

   named_cnd.notify_all(); //다른 어플이 영원히 기다리는것 방지.

//cleanUp...
   boost::interprocess::shared_memory_object::remove("shm");
   boost::interprocess::named_mutex::remove("mtx");
   boost::interprocess::named_condition::remove("cnd");
}

2개의 어플리케이션을 동작시키면, 먼저 실행된 쪽에서 짝수를 출력하게 된다.
조건변수는 다른 어플리케이션에게 작업이 완료되었음을 알리기 위한 목적으로 사용된다.
조건변수 사용을 위해, boost::interprocess::named_condition 사용되고 있다.
동기화를 위해 wait(), notify_all() 메서드가 사용되는데,
wait() 호출시 넘겨준 mutex object는 먼저 해제되고, 다른 어플에서 notify_all() 메서드
를 호출할때까지 블럭된다.

앞서 설명된, mutex에 2가지 종류-즉,anonymous type과 named type- 가 있는것처럼
조건변수에도 2가지 유형이 존재한다. (즉, 방금전의 예제는 named_condition 이었다)

다음은 앞서의 예제를 anonymous조건변수 사용으로 수정한것이다.

#include <boost/interprocess/managed_shared_memory.hpp>
#include <boost/interprocess/sync/interprocess_mutex.hpp>
#include <boost/interprocess/sync/interprocess_condition.hpp>
#include <boost/interprocess/sync/scoped_lock.hpp>
#include <iostream>

int main()
{
   try
{
boost::interprocess::managed_shared_memory
     managed_shm(boost::interprocess::open_or_create, "shm", 1024);
    int *i = managed_shm.find_or_construct<int>("Integer")(0);
    boost::interprocess::interprocess_mutex *mtx =
     managed_shm.find_or_construct<boost::interprocess::interprocess_mutex>("mtx")();

    boost::interprocess::interprocess_condition *cnd =
     managed_shm.find_or_construct<boost::interprocess::interprocess_condition>("cnd")();

    boost::interprocess::scoped_lock<boost::interprocess::interprocess_mutex> lock(*mtx);

    while (*i < 10)
    {
       if (*i % 2 == 0)
       {
         ++(*i);
         cnd->notify_all();
         cnd->wait(lock);
       }
       else
       {
         std::cout << *i << std::endl;
         ++(*i);
         cnd->notify_all();
         cnd->wait(lock);
       }
    }
    cnd->notify_all();
   }
   catch (...)
   {
   }
   boost::interprocess::shared_memory_object::remove("shm");
}
